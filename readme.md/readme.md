RELIANCE CONSTRUCTION COMPANY TEMPLATE
=======================================

HOW TO USE
----------
1. Unzip the folder.
2. Open index.html in any modern browser.
3. No installation required.

FILE STRUCTURE
--------------
index.html         → Main homepage
assets/css/        → Custom styles
assets/js/         → JavaScript file
assets/images/     → Images (logo + placeholders)

REPLACE CONTENT
---------------
Logo:
assets/images/logo.jpg

Hero Background:
assets/images/hero.jpg

Service Images:
assets/images/service1.jpg
assets/images/service2.jpg
assets/images/service3.jpg

Text:
Open index.html and replace placeholder content inside:
- Hero section
- About section
- Services section
- Contact section

COLORS
------
Main colors are defined in:
assets/css/style.css
inside :root variables.

RESPONSIVE
----------
Uses Bootstrap 5 grid system.
Works on desktop, tablet, and mobile with one unified layout.
